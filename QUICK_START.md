# ⚡ Быстрый старт

## Запуск за 3 шага

### 1. Запустите Docker Compose

```bash
docker-compose up --build
```

### 2. Дождитесь запуска всех сервисов

Вы увидите сообщения о том, что:
- ✅ PostgreSQL запущен
- ✅ Backend запущен на порту 3001
- ✅ Frontend запущен на порту 3000

### 3. Откройте браузер

Перейдите на http://localhost:3000

## Первые шаги

1. **Зарегистрируйтесь** как студент или преподаватель
2. Если вы **преподаватель**:
   - Создайте курс (функционал в разработке, но API готов)
   - Добавьте материалы и задания
3. Если вы **студент**:
   - Запишитесь на курс (через API или когда будет готов UI)
   - Просматривайте материалы
   - Отправляйте задания

## Добавление логотипа

Поместите файл `logo.png` в `frontend/public/logo.png`

## Остановка

```bash
docker-compose down
```
