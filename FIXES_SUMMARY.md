### 3. Пример SCORM файла
**Проблема:** Не было примера для тестирования SCORM
**Решение:** Создан полный пример SCORM 1.2 пакета
**Файлы:**
- `SCORM_EXAMPLE/imsmanifest.xml` - манифест SCORM
- `SCORM_EXAMPLE/lesson1.html` - первый урок
- `SCORM_EXAMPLE/lesson2.html` - второй урок с тестом
- `SCORM_EXAMPLE/scorm_api_wrapper.js` - обертка для SCORM API
- `SCORM_EXAMPLE/README.md` - инструкция по использованию

**Как использовать:**
1. Создайте ZIP архив из всех файлов в папке `SCORM_EXAMPLE`
2. В системе создайте новый материал типа "SCORM"
3. Загрузите ZIP архив
4. Система автоматически распакует и установит пакет

**Особенности примера:**
- Два урока с разным контентом
- Интерактивный тест во втором уроке
- Сохранение прогресса через SCORM API
- Работает локально (через localStorage) для тестирования
- Полностью совместим с SCORM 1.2 стандартом

## Структура чата

Чат организован следующим образом:
- **Курс** = Группа (все участники курса)
- **Тема** = Отдельное обсуждение внутри курса
- **Сообщения** = Сообщения внутри темы

Преподаватель и студенты могут:
- Создавать темы
- Отправлять сообщения
- Удалять свои сообщения
- Видеть все сообщения в курсе

## Тестирование

### Чат
1. Зайдите как преподаватель или студент
2. Откройте курс
3. Перейдите на вкладку "Чат"
4. Создайте тему и отправьте сообщение

### SCORM
1. Создайте ZIP из файлов в `SCORM_EXAMPLE/`
2. Загрузите как SCORM материал в курс
3. Откройте материал - уроки должны загрузиться
4. Пройдите уроки и проверьте сохранение прогресса

## Примечания

- Чат обновляется автоматически каждые 3 секунды
- SCORM пример работает локально через localStorage (для тестирования)
- В реальной LMS системе SCORM API предоставляется автоматически
