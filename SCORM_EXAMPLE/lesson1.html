Урок 1: Введение в SCORM

Что такое SCORM?

SCORM (Sharable Content Object Reference Model) - это стандарт для электронного обучения, который позволяет создавать переиспользуемые учебные материалы.

Основные возможности:

  • Отслеживание прогресса обучения
  • Сохранение результатов
  • Совместимость с различными LMS системами
  • Интерактивность

Как это работает?

SCORM использует API для связи между контентом и системой управления обучением (LMS). Это позволяет отслеживать, что студент изучил материал и сохранить его прогресс.
