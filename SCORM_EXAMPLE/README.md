# Пример SCORM пакета

Это простой пример SCORM 1.2 пакета для демонстрации работы с SCORM контентом в системе.

## Структура пакета

```
SCORM_EXAMPLE/
├── imsmanifest.xml      # Манифест SCORM (обязательный файл)
├── lesson1.html         # Первый урок
├── lesson2.html         # Второй урок с тестом
├── scorm_api_wrapper.js # Обертка для SCORM API
└── README.md            # Этот файл
```

## Как использовать

### Вариант 1: Загрузка в систему

1. Создайте ZIP архив со всеми файлами из папки `SCORM_EXAMPLE`
2. В системе управления курсом выберите "Создать материал"
3. Выберите тип "SCORM"
4. Загрузите ZIP архив
5. Система автоматически распакует и установит SCORM пакет

### Вариант 2: Тестирование локально

1. Откройте `lesson1.html` или `lesson2.html` в браузере
2. Функции SCORM будут работать через localStorage (заглушка)
3. В реальной LMS системе SCORM API предоставляется автоматически

## Особенности

- **imsmanifest.xml** - описывает структуру курса, уроки и ресурсы
- **lesson1.html** - простой урок с кнопкой завершения
- **lesson2.html** - интерактивный урок с тестом
- **scorm_api_wrapper.js** - обертка для связи с LMS системой

## SCORM API методы

Пакет использует следующие методы SCORM API:

- `LMSInitialize()` - инициализация сессии
- `LMSSetValue()` - сохранение данных (прогресс, оценка, статус)
- `LMSGetValue()` - получение сохраненных данных
- `LMSCommit()` - сохранение изменений
- `LMSFinish()` - завершение сессии

## Данные, которые сохраняются

- `cmi.core.lesson_status` - статус урока (completed, passed, failed)
- `cmi.core.score.raw` - полученные баллы
- `cmi.core.score.max` - максимальные баллы

## Примечания

- В реальной LMS системе SCORM API предоставляется автоматически
- Локальное тестирование использует localStorage как заглушку
- Для полной функциональности загрузите пакет в систему управления обучением

## Создание ZIP архива

Для загрузки в систему создайте ZIP архив:

```bash
cd SCORM_EXAMPLE
zip -r scorm_example.zip imsmanifest.xml lesson1.html lesson2.html scorm_api_wrapper.js
```

Или используйте любой архиватор (WinRAR, 7-Zip и т.д.) для создания ZIP файла со всеми файлами из папки.
